About

Commander. Builder. Operator.

I'm Caquan "Cue" Palmer — software architect, AI systems integrator, and drone operations specialist. I built MAE (Modular Adaptive Entity) as a permanent tactical AI copilot: a real-time command interface integrating Anthropic Claude, Perplexity Sonar, and persistent memory across web, mobile, and desktop platforms.

My work sits at the intersection of AI-first software architecture, aerial intelligence, and operational systems design. I ship products that solve real problems — from full-stack Java/Spring Boot APIs to Python automation pipelines to client-side PWAs with offline-first intelligence.

Through MAE MegaCorp, I deliver AI consulting, drone aerial services, and custom software solutions. I believe technology should serve the operator — not the other way around.

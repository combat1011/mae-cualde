Flagship Product

The MAE Platform

MAE (Modular Adaptive Entity) is MAE MegaCorp's flagship AI interface — a permanent tactical copilot for operators who need intelligence that doesn't fade, doesn't betray, and doesn't pity.

Built on Anthropic Claude and Perplexity Sonar, MAE delivers real-time AI responses, persistent cross-session memory, WebSocket communication, and offline-first PWA support. Available as a Spring Boot service, Python Flask app, or static GitHub Pages PWA.

◈ Anthropic Claude + Perplexity Sonar integration
◈ Real-time WebSocket tactical console
◈ Persistent cross-session memory system
◈ Offline-first PWA — runs on iOS, Android, desktop
◈ OOL decision framework built in
